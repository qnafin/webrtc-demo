WebRTC, передача SDP без сигнализации.

Инструкции: Начните с открытия двух вкладок рядом и следуйте инструкциям ниже, чтобы передать предложение SDP и ответ. Я буду называть каждую вкладку User 1 и User 2.

Source Code

Шаг 1: User 1, нажмите кнопку «Создать предложение». чтобы создать предложение SDP и скопировать предложение из текстовой области ниже.

Шаг 2: User 2, pвставьте предложение SDP, созданное пользователем 1, в текстовую область выше, затем нажмите «Создать ответ». чтобы сгенерировать ответ SDP, скопируйте ответ из текстовой области ниже.

Step 3: User 1, вставьте предложение SDP, созданное пользователем 2, в текстовую область выше, а затем нажмите «Добавить ответ»



Step 4: Отправьте ошибки, любым удобным способом
